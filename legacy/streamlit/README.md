# Legacy Streamlit prototype

This is the **original** Probability Paradoxes app, built as a single Streamlit
project. It has been superseded by the FastAPI + Next.js stack at the repository
root, but is kept here because it still runs and is a compact reference for the
simulation logic.

## Run it

```bash
cd legacy/streamlit
pip install -r requirements.txt
streamlit run app/app.py
```

See `docs/structure.md` for the original architecture notes.
